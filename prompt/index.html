    Quick Start

    1. Write your prompt template in the editor using {{variable_name}} for dynamic values.

    2. Fill in variable values in the right panel - they auto-detect as you type!

    3. Preview in different formats (Plain, OpenAI, Anthropic).

    4. Test your prompt with real inputs in the Test & Iterate tab.

    5. Export your finalized prompt in any format.

    Variable Syntax

    Variables are defined using double curly braces: {{variable_name}}

    Variable names can contain letters, numbers, and underscores. They'll be highlighted in purple/magenta!

    Best Practices

    • Be specific and clear in your instructions
    • Include examples when possible
    • Define the desired tone and format
    • Add constraints to guide the AI
    • Test with multiple inputs to ensure robustness

    Keyboard Shortcuts

    Ctrl+S: Save template (auto-saves every 2 seconds)

    Ctrl+L: Open template library

    Ctrl+E: Focus on template editor

## 📋 Troubleshooting

### **"I didn't receive an email"**
1. Check spam/junk folder (FormSpree emails sometimes land there)
2. Confirm email: `hwmoses2@icloud.com` is correct
3. Test the form yourself: Fill it out and submit
4. Wait 1-2 minutes (sometimes slight delay)

### **"Form says error"**
- FormSpree free tier = 50 submissions/month
- If exceeded, upgrade to paid ($10/mo)
- Check FormSpree status: https://status.formspree.io

### **"Want to change email address"**
- Create new FormSpree form with new email
- Update `action` URL in index.html
- Commit and push changes

### **"Form not showing on site"**
- Clear browser cache (Ctrl+Shift+R / Cmd+Shift+R)
- Check GitHub Pages deploy (2-5 minute delay)
- Verify commit: https://github.com/hmoses/poly-glot/commits/main

# Changelog

All notable changes to Poly-Glot VS Code Extension.

## [1.0.0] — 2025-03-22

### Added
- 🎉 Initial release
- `polyglot.generateComments` — Generate inline AI comments (`Cmd+Shift+/`)
- `polyglot.explainCode` — Deep code analysis panel (`Cmd+Shift+E`)
- `polyglot.configureApiKey` — Secure API key setup via SecretStorage
- `polyglot.openTemplates` — Templates sidebar
- 12-language support: JavaScript, TypeScript, Python, Java, C#, C/C++, Go, Rust, Ruby, PHP, Swift, SQL
- OpenAI and Anthropic provider support
- Per-request cost display in status bar
- Templates sidebar with copy-to-clipboard
- Context menu integration (right-click → generate/explain)
